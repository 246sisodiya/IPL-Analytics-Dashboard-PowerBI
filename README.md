

#  IPL Analytics Dashboard | Power BI Project

##  Project Overview

The IPL Analytics Dashboard is an end-to-end data analytics project developed using Power BI to analyze historical Indian Premier League (IPL) data.

This dashboard provides comprehensive insights into team performance, player statistics, venue analysis, season trends, and match-related metrics through interactive visualizations and dynamic filtering capabilities.

The objective of this project is to transform raw IPL data into meaningful business insights that help users understand team strategies, player contributions, venue performance, and overall tournament trends.

---

##  Project Objectives

- Analyze IPL performance across multiple seasons.
- Evaluate team-wise and player-wise statistics.
- Compare venue performance and winning patterns.
- Identify top-performing batsmen and bowlers.
- Track season-wise trends in scoring and match outcomes.
- Build an interactive dashboard for data-driven decision-making.

---

##  Dashboard Modules

### 1. IPL Overview

This section provides a high-level overview of the tournament, including:

- Total IPL Seasons
- Total Teams Participated
- Total Matches Played
- Average Runs Per Match by Season
- No Ball and Wide Ball Analysis
- Stadium Ranking Analysis
- Most Used TV Umpires
- Playoff Qualification History

---

### 2. Team Profile Analysis

This section enables detailed analysis of individual IPL teams.

Key insights include:

- Team-wise Winning Percentage
- Venue-wise Performance
- Matches Played, Won and Lost
- Season-wise Team Performance
- Top Run Scorers
- Top Wicket Takers
- Boundary Analysis
- Performance Comparison Across Seasons

Interactive filters allow users to analyze specific teams and seasons.

---

### 3. Player Profile Analysis

This section focuses on player-level performance analytics.

Features include:

- Batsman vs Bowler Analysis
- Runs Scored
- Strike Rate Analysis
- Wickets Taken
- Economy Rate Analysis
- Top Run Scorers of Selected Season
- Top Wicket Takers of Selected Season
- Top Boundary Scorers

Users can dynamically filter players and seasons for detailed performance evaluation.

##  Data Source

The dataset used in this project was obtained from Kaggle and contains historical IPL match, player, team, venue, and season-wise information.

Data Source: Kaggle IPL Dataset

The raw data was further cleaned, transformed, and modeled using Power Query and Power BI before creating the final dashboard.

##  Key Analytical Features

### Dynamic Filtering
Users can filter data based on:

- IPL Season
- Team Name
- Batsman
- Bowler
- Performance Metrics

### Interactive Visualizations

The dashboard includes:

- KPI Cards
- Line Charts
- Bar Charts
- Comparative Analysis Charts
- Performance Tables
- Dynamic Slicers

### Advanced DAX Measures

Custom DAX calculations were used to derive:

- Winning Percentage
- Venue Success Rate

- Player Rankings
- Strike Rate Metrics
- Economy Rate Metrics
- Season Performance Indicators
